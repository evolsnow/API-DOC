# 处理消息



## 接口说明

　标志消息已处理/未处理

## 


* **URL**
        /chats/status/:cid
        

* **Method:**
  
  ####<font color=orange>`PUT`</font>

* **权限:**

  `medium`

*  **请求参数**

**键值** | **类型** | **是否必需** | **备注**
---------|----------|--------------|---------
dealt|bool|是|是否已处理

* **请求内容示例:**


        {
            "dealt": true
        } 
--- 
*  **返回参数说明**

    基本返回
