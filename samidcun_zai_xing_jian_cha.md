# 获取samId可用状态

## 接口说明

　检查samId是否可用

## 


* **URL**
        /samIds/:samId

* **Method:**
  
  ####<font color=blue>`GET`</font>

* **权限:**

  `low`

*  **请求参数**

**键值** | **类型** | **是否必需** | **备注**
---------|----------|--------------|---------
samId|string|是|sam id

* **请求内容示例:**


        /samIds/mySamId
----------------------- 
*  **返回参数说明**

**键值** | **类型** | **是否必需** | **备注**
---------|----------|--------------|---------
code    |int |是 |请求结果
msg      |string|否 |不可用原因

* **返回成功示例:**


        {
            "code": 1,
            "msg": "已被使用/非法字符..."
        }
